## Tech Stack

Built with vanilla HTML, CSS, and JavaScript. No frameworks, no build tools, no dependencies. Just one file that loads instantly anywhere.

- **Layout:** CSS Grid · Flexbox · Custom properties for theming
- **Typography:** Share Tech Mono · Outfit (Google Fonts)
- **Animations:** Pure CSS keyframes · IntersectionObserver for scroll reveal
- **Interactive:** SOC Analyst HUD · Floating action dock · Keyboard shortcuts modal
- **Theme:** Custom hacker blue palette with subtle scanline overlay

## Running Locally

Clone and open. Nothing else.

```bash
git clone https://github.com/TheCaptainCJ/TheCaptainCJ.github.io.git
cd TheCaptainCJ.github.io
# Open index.html in any browser
```

Or with a quick Python server if you prefer:

```bash
python -m http.server 8000
# Visit http://localhost:8000
```
