PROJECT 08
Enterprise Homelab SIEM
Splunk · Active Directory · pfSense · Kali · SOC Environment
Built a fully functional SOC environment on one physical machine using nested virtualization. Proxmox hosts pfSense for VLAN segmentation, a Windows Server domain controller, a domain-joined workstation, Kali Linux on an isolated attack VLAN, and Splunk Enterprise as the SIEM. Deployed Universal Forwarders on both Windows machines to ship Security, System, and Application event logs into Splunk on TCP 9997. The whole lab is designed for attack simulation and detection engineering practice.

PROJECT 09
SOC Case Study: Phishing Investigation
Splunk SIEM · Alert Triage · False Positive · Incident Report
Investigated a medium-severity phishing alert (Alert 8814) triggered by a SIEM rule detecting an inbound email with an external link. Worked through the full SOC triage workflow: reviewed sender domain, recipient context, email content, and embedded URL.
